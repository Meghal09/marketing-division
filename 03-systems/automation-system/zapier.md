# Zapier Automation System

## Purpose
To automate repetitive marketing tasks using Zapier.

---

## Use Cases
- Lead capture → CRM  
- Form submissions → Email sequences  
- Social posts → Database logging  
- Funnel events → Notifications  

---

## Workflow
1. ATHENA‑MK defines automation logic  
2. ECHO‑EM builds email flows  
3. Automation triggers run in Zapier  
4. CRONOS‑AN monitors performance
