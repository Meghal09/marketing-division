# Marketing Division Workflows

This document defines how tasks flow through the FORMA Marketing Intelligence Division (FMID).

---

# 1. Content Production Workflow

1. ATHENA‑MK creates the content strategy  
2. MUSE‑CG generates scripts and ideas  
3. ZEUS‑TT produces short‑form content  
4. ORPHEUS‑CW writes long‑form content  
5. IRIS‑SM schedules posts  
6. CRONOS‑AN tracks performance  

---

# 2. Funnel Workflow

1. ATHENA‑MK designs funnel structure  
2. JANUS‑FB builds funnel pages  
3. ORPHEUS‑CW writes copy  
4. ARES‑ADS drives traffic  
5. CRONOS‑AN monitors conversions  

---

# 3. Automation Workflow

1. ATHENA‑MK defines automation logic  
2. ECHO‑EM builds email sequences  
3. Automation System triggers workflows  
4. CRONOS‑AN measures engagement  

---

# 4. Launch Workflow

1. PROMETHEUS‑MK approves launch  
2. ATHENA‑MK creates launch plan  
3. ZEUS‑TT + MUSE‑CG produce content  
4. ARES‑ADS runs paid campaigns  
5. IRIS‑SM handles social distribution  
6. CRONOS‑AN reports results  

---

# 5. Reporting Workflow

1. CRONOS‑AN collects data  
2. Dashboards update automatically  
3. APOLLO‑MK reviews performance  
4. PROMETHEUS‑MK adjusts strategy
